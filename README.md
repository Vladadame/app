Symfony App Boilerplate

This repository is a basic Symfony application starter template.

📌 Purpose

This project was created as a foundation for future development.
It is not a complete application and does not represent a finished product.

The goal of this repository is to:

provide a ready-to-use Symfony setup
include Docker-based development environment
serve as a starting point for new features and experiments
⚙️ Tech Stack
PHP (Symfony)
PostgreSQL
Docker / Docker Compose
Webpack Encore (frontend assets)
🚧 Current State

At the moment, this project includes only:

basic Symfony structure
minimal controller and template setup
default configuration files

No business logic or production-ready features are implemented yet.

📈 Future Plans

This repository may be used to build:

API services
web applications
experimental features
pet-projects
⚠️ Notes
This is a work in progress / starter template
Not intended for production use in its current state

Feel free to use this as a base for your own Symfony projects.
